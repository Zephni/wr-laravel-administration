Welcome to WRLA Documentation

WebRegulate Laravel Administration (WRLA) is a powerful administration panel package for Laravel applications. This documentation will guide you through installation, configuration, and advanced usage.

Installation

Get started by installing WRLA in your Laravel application.

Configuration

Configure WRLA to match your application's needs.

Manageable Models

Learn how to create and manage models in the admin panel.

Authentication

Set up authentication and user management.
Requirements: Laravel v11 or higher is required to use WRLA.

Key Features

  • Easy installation with a single artisan command
  • Intuitive model management interface
  • Built-in authentication and permissions system
  • Highly customizable and extensible
  • Modern, responsive UI with Tailwind CSS
