Installation

Important: Note that you MUST use at least Laravel v11 to use webregulate/laravel-administration.

1 Require the package with composer 

composer require webregulate/laravel-administration "dev-main"

2 Install with the built-in command 

php artisan wrla:install

Follow the steps provided by the installation wizard. Note that installing will publish a few assets which will be added to your project and mentioned in your terminal. You will also be asked if you want to run the migrations, and also whether you like to create a master user, which are necessary to use WRLA.

3 Access the administration section

You can now access the administration section by heading to: 

{your app url}/wr-admin

4 Configure the package

While installing, WRLA will create a config file at config/wr-laravel-administration.php. Feel free to edit this as you wish. One of the first things you may want to change is the route URL for the admin section itself, which is the first key => value in the configuration file: 

'base_url' => 'wr-admin',

5 Add Manageable Models

Now you are ready to add Manageable Models. You can either create new models to manage within your new administration section or use them to setup models that already exist in your project. 

php artisan wrla:manageable-model {?ModelName}

For more information on Manageable Models, see the Manageable Models documentation.

Note: If you wish to create other users from your terminal you can run the appropriate artisan command.
